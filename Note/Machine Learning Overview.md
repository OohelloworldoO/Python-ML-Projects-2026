# Machine Learning Overview

## What is Machine Learning?

Machine learning aims to learn a function from data.

---

## ML Tasks

- Regression
- Classification
- Structured prediction (optional)

---

## ML Pipeline

1. Define problem
2. Collect data
3. Define model
4. Define loss
5. Optimization
6. Evaluation

---

## Flow

Data → Model → Loss → Optimization → Prediction

---

## Key Components

### 1. Model

Define the function that maps input to output  
👉 Details: ../2_Core_Concepts/Model.md

---

### 2. Loss Function

Measure how wrong the prediction is  
👉 Details: [Loss_Function.md](./Note/Loss_Function.md)

---

### 3. Optimization

Update parameters to minimize loss  
👉 Details: [Optimization.md](./Note/Optimization.md)

---

### 4. Activation Function (Neural Networks)

Introduce non-linearity into the model  
👉 Details: [Activation_Function.md](./Note/Activation_Function.md)

---

## Key Idea

- Model defines hypothesis space
- Loss measures error
- Optimization finds best parameters
